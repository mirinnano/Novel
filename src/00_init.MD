# 00_init.MD

彼女が余命宣告されたのは、鬱陶しいほど日差しが強く降り注ぐ日のことだった。


# 回想

俺たちは、親友ですらなかった。
出席番号が近かったから一度だけ掃除当番を代わったことがあるだけの、取るに足らないクラスメイトだった。

彼女が数週間学校を休んでも、俺は「風邪でも長引いてるのか」としか思わなかった。

だから、あの日。

校舎裏で彼女に呼び止められたとき、最初は告白かと思った。
でも、そんな顔じゃなかった。

少女「ねえ、伝えないといけないことがあるんだ。」

その表情で、俺はすぐに察した。
良い話じゃない。

俺「.....どうした。」

少女「もう私、長くないみたい。」

少女「お医者さんに、そう言われちゃったよ。」

セミの声だけがやかましい。

俺は彼女の目を見られなかった。見る必要がなかった。

少女「びっくりするよね。あんまり仲良くない女の子に急に話しかけられて、『もう長くない』なんて言われたらさ。」

彼女の顔に、涙はなかった。
まるで最初から、その呪われた運命を静かに受け入れたように......

俺「………」

少女「そんな怖い顔しないでよ。まだ死ぬわけじゃないんだし。」

俺「だとしても……」

いつかは死ぬ。それだけは確かだ。

少女「心配してくれて嬉しいよ。なんてね。」

沈黙。

俺「……じゃあ、あの話はどうなる。」

少女「どの話？」

俺「旅行だよ。ずっと授業中にノートに書いてたからさ、少し気になって。」

少女「……あぁ、あのノートのことか。」

少女「授業中に、一人でノートに書いてたよね。北海道の深い森に行って、駅弁のいかめし食べたいとか……」

彼女はそこで言葉を切った。
視線が、自分の影に落ちる。

少女「……でも、もういいよ。そんなのただの夢だもん。先生も絶対ダメだって言うだろうし。私、もうすぐ部屋から出られなくなっちゃうから。」

少女の表情は変わらず、無機質に俺の問いに答える。

俺「だったら。」

俺はセミを黙らせるように、語りだす。

俺「俺が、お前の最後の旅に付きやってやる」

彼女を救いたかったわけでも、同情したわけでもない。
ただ、このどうしようもなく単調な日々に、変化を起こしたかったのだろう。


少女「……えっ？」

俺「まだ死なないんだろ？」

少女は一瞬だけ目を丸くして、それから、今まで一番人間らしく笑った。

少女「うん……」

俺「じゃあ、行くぞ。」
